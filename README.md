# 📄 AI Notes Summarizer

A simple AI-powered web app that summarizes study notes using Google Gemini API.

---

## 🚀 Features

* Paste notes and get instant summary
* Multiple summary styles (brief, detailed, bullet points)
* Clean and simple UI

---

## 🛠️ Tech Used

* HTML
* CSS
* JavaScript
* Google Gemini API

---

## ▶️ How to Run

1. Download or clone this repository
2. Open `app.js`
3. Add your API key:

   ```js
   const API_KEY = "YOUR_API_KEY_HERE";
   ```
4. Open `index.html` in browser

---




This project uses a pre-trained AI model via API. No model training is done locally.
